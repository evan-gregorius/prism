# Intended Use Statement

PRISM is an early-stage research-software project for studying fast forward and inverse engineering models.

The repository is intended for research, education, reproducibility, portfolio development, and possible future collaboration. It does not currently provide validated performance and should not be used for clinical decisions, safety-critical work, operational decisions, or any other use where an incorrect result could cause harm.

Each application must document its own assumptions, supported parameter range, validation evidence, known limitations, and intended-use boundaries.

Public availability does not imply that the software is complete, generally applicable, approved by any institution, or licensed for unrestricted reuse. Users are responsible for reviewing the repository license and the licenses of all data, software, and external sources before reuse.
